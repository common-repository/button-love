== Description ==

The software uses a shortcode to place buttons within your content. 

== Usage ==

Shortcode format:  
[buttonlove link="link-here" color="blue"]Button Text Here[/buttonlove]

Atributes:
link: The link that user will be taken to when they click the button.
color: The color of the button.  Available colors: orange, blue, green, red.
